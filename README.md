# MapleSoundz

MapleSoundz is a small browser soundboard for MapleStory-inspired UI and event sounds.

## Run locally

```bash
npm start
```

The app will be available at `http://127.0.0.1:3000`.

## Build the single-file version

```bash
npm run build
```

This generates a standalone HTML file at `dist/maplesoundz-single.html`.
